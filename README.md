# home-web-hub

一个用于归档和发布多个独立 HTML 页面的仓库。

## 项目简介

本项目旨在集中存放各类自制的 HTML 页面，方便在不同场景下查看、测试或引用。每个页面均为独立的静态文件，不依赖外部服务或特定域名。

## 目录说明

```
home-web-hub/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的资源文件（如 CSS、JS、图片等）
├── docs/           # 相关文档或说明
└── README.md       # 本文件
```

- **pages/**：每个 HTML 文件均为一个独立页面，命名应简要反映其用途。
- **assets/**：若页面需要额外资源，请放置在此目录下，并按页面名称组织子目录。
- **docs/**：可存放页面使用说明、设计思路等补充文档。

## 页面归档说明

- 所有页面均以 `.html` 后缀命名，存放于 `pages/` 目录下。
- 页面应保持自包含，尽量减少外部依赖；如有必要，请将资源文件放入 `assets/` 对应子目录。
- 每个页面建议在头部注释中注明创建日期、简要描述及作者（可选）。
- 本仓库不承诺对页面内容进行定期更新或维护，归档后即视为稳定版本。

## 维护说明

- 欢迎通过 Pull Request 提交新的 HTML 页面或改进现有页面。
- 提交前请确保页面可在主流浏览器中正常运行，且不包含恶意代码或外部跟踪脚本。
- 对于不再需要的页面，可通过 Issue 讨论是否移除，但建议保留历史版本。
- 本仓库遵循开源精神，但不对页面内容的准确性、适用性作任何保证。
